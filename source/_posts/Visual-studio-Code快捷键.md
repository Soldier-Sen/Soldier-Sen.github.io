# 跳转定义返回快捷键
windows系统:
Alt+← .. navigate back
Alt+→ .. navigate forward

Mac系统:
Ctrl+- .. navigate back
Ctrl+Shift+- .. navigate forward

On Ubuntu Linux系统:
Ctrl+Alt+- .. navigate back
Ctrl+Shift+- .. navigate forward
